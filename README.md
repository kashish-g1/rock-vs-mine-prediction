# Rock vs Mine Prediction using Machine Learning

This machine learning project uses Logistic Regression on the Sonar dataset to classify objects as Rock or Mine.

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Jupyter Notebook

## Model Used
- Logistic Regression

## Accuracy
- Training Accuracy: 83%
- Testing Accuracy: 76%

## Project Files
- model.ipynb → complete notebook
- Copy of sonar data.csv → dataset

## Output
The model predicts whether an object is:
- Rock (R)
- Mine (M)
